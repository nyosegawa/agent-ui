# Agent UI Docs Site Example

Small executable package overview and demo shell for Agent UI.

Detailed docs: [docs/examples/docs-site.md](../../docs/examples/docs-site.md).

This example intentionally remains executable. It is a lightweight package
overview and compile/style smoke surface for the published React stylesheet and
components, not the markdown documentation site and not a host runtime.

Run:

```sh
bun --filter @nyosegawa/agent-ui-docs-site dev
```
