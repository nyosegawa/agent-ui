# Archive

This directory keeps historical planning and validation records that are useful
for project archaeology but should not be read as the current implementation
contract.

- `2026-vnext-plan.md`: original vNext migration plan.
- `2026-vnext-completion-log.md`: original milestone checklist and completion log.
- `testing-history-2026-05.md`: dated validation reports from the May 2026 buildout.
- `completeness-audit-2026-05.md`: historical completeness audit snapshot.
- `pre-vnext-roadmap.md`: roadmap snapshot from before the current
  transcript-first architecture stabilized.

Current docs live one level up in `docs/`; current direction and open work live
in repository-root `PLAN.md` and `TODO.md`.
